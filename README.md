## Specifications

Language : python
Framework: Flask
Dependencies:
- flask
- pydotenv

Endpoints:
- '/' or '/index' or '/home' : Landing Page
- '/coins' : Task Page

## Installation

```
git clone <link-to-repo>
cd <repo>
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Execution

```
flask run -p 8080
```
